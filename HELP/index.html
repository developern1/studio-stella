Stella Studio Guide

Stella Studio는 오디오 파일의 cut편집과 faidin, faidout을 지원하는 웹 어플리케이션입니다.
사용법을 간단하게 설명합니다. 아래의 순서 대로 작업해 주세요.

1. 편집할 파일을 업로드 합니다.

upload 버튼을 클릭하면 탐색기가 나타납니다. 여기서 편집할 파일을 선택해 주세요.


파일이 로딩되는 동안 인디케이터가 표시됩니다.



2. cut 시작점 지정

파일이 로딩되면 파형과 함께 모든 메뉴 아이콘이 활성화됩니다.
그리고 화면 하단에 오디오 풀레이어가 표시됩니다.
이 프레이어를 이용하여 원하는 위치를 찾아주세요.
마우스로 원하는 위치를 클릭하면 시작점이 지정됩니다.



3. split 버튼으로 시작점 cut

split 버튼을 클릭하면 시작점을 기준으로 오디오 파형이 둘로 나뉘어집니다.



첫 번째 파형은 cut할 부분이고, 두 번째 파형은 원하는 파형의 시작점이 됩니다.



3. 앞부분 파형 삭제

첫 번쨰 파형의 왼쪽 x 버튼을 클릭하면 앞부분 파형이 삭제됩니다.



첫 번째 파형이 삭제되면 두 번째 파형이 첫 번째 파형이 됩니다.
시작 부분을 선택할 때와 마찬가지로 마우스로 원하는 위치를 클릭하면 끝나는 지점이 지정됩니다.
다시 split 버튼을 클릭하면 두 번째 파형이 분리 됩니다.



4. 필요없는 분리된 파형 삭제

앞부분 삭제할 때와 마찬가지로 두 번째 파형의 x표를 클릭하여 필요없는 뒷부분 파형을 삭제합니다.



5. 파형의 위치 변경

4번까지의 과정을 거치면 원하는 파형이 완성됩니다.
만일 재생버튼을 클릭하여 오디오를 들어보고 조금 더 수정이 필요하다면 2~4번의 과정을 반복합니다.
원하는 파형이 완성되면 shift 버튼을 클릭하고 파형을 드래그하여 원하는 위치로 이동시킵니다.

6. Faidin 작업

Faidin은 오디오가 점점 크게 들리도록 하는 효과입니다.
먼저 faidin 버튼을 클릭합니다.
그리고 파형의 시작부분을 오른쪽으로 드레그하면 faidin이 적용됩니다. 얼마나 드레그하냐에 따라 faidin 효과가 결정됩니다.
오디오를 들어보고 조정이 필요하다면 다시 드레그해서 조정합니다.




7. Faidout 작업

Faidout은 오디오가 점점 작게 들리도록 하는 효과입니다.
먼저 faidout 버튼을 클릭합니다.
그리고 파형의 끝부분을 왼쪽으로 드레그하면 faidout이 적용됩니다. 얼마나 드레그하냐에 따라 faidout 효과가 결정됩니다.
오디오를 들어보고 조정이 필요하다면 다시 드레그해서 조정합니다.

8. 완성된 파형 앞으로 옮기기

완성된 파형은 제일 앞으로 옮겨야 합니다.
왜냐하면 그대로 저장하면 앞쪽의 빈 부분은 묵음이 되기 때문입니다.
따라서 shift 버튼을 클릭하고 완성된 파형을 드래그하여 제일 앞으로 옮깁니다.
파형 아래쪽에 있는 스크롤 버튼을 같이 이용하여 이동시키면 편리합니다.



파형을 앞쪽으로 이동했으면 cursor 버튼을 클릭한 후 아래쪽의 플레이 버튼을 클릭하여 오디오가 원하는 대로 잘 들리는지 확인합니다.



9. 완성된 오디오 다운로드

오디오가 완성되었으면 download 버튼을 클릭하여 오디오를 다운로드 받습니다.
